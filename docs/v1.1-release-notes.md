# PredictX Sports v1.1 更新事項

## 1. 歷史賽事線上刷新功能補強
- **問題**：歷史賽事下拉刷新因 `hasLoadedFullHistory` 一次性快取護欄而無實際 API 請求，需重啟 App 才能取得最新結算資料。
- **修正**：移除 `HomeStore.loadHistoryForLeague()` 中的 `hasLoadedFullHistory` 守衛邏輯，使每次下拉刷新均重新向後端 `/api/games` 發送請求並更新 `historicalMatches`。
- **驗證**：下拉刷新 HistoryView 後，`aiIsHit`、`homeScore`/`awayScore` 等欄位應與後端即時同步，無需重啟 App。

## 2. 新增 WNBA 賽事
- 在聯賽清單中新增 WNBA 支援
- 涵蓋範圍：賽事列表、AI 分析預測、歷史賽事結算、命中率統計
- 需要配合後端 ingester（新增 WNBA 資料來源）、前端 LeagueType enum 擴充、API 合約確認
