## 這是什麼 App？

PredictX Sports 是一個**球類比賽 AI 推論 App**。我們用雲端 LLM 分析每日 MLB / NPB / CPBL / NBA 賽事，給出：

- **推論機率**（兩隊勝率）
- **預測比分**（AI 認為最終比數）
- **AI 分析摘要**（200-400 字的球評風格分析）
- **雷達圖**（6 維度球隊對比）
- **歷史驗證率**（AI 過去預測命中率）

**這是純資訊 App，不接受投注、不賣預測商品**。會員訂閱是 4 級制（月費 80/160/240/320 TWD 與年費方案），免費版可看每日 3 場、付費解鎖全部。

## 測試任務清單（請每項都試）

### 0. 登入 / 註冊

- [ ] 開啟 App，看首頁是否正常載入賽事列表
- [ ] 切到「個人」tab，看會員等級（預設應該是 Free）
- [ ] 點「升級方案」看 4 級會員方案
- [ ] **不要實際購買**，但可以截圖給我看方案呈現是否正確

### 1. 賽事列表

- [ ] 確認今天日期顯示是「06/22 週一」這類 MM/dd 格式（**不是** 06-22 2026）
- [ ] 確認至少有 MLB 賽事卡片顯示
- [ ] 確認 AI 推論機率百分比有顯示（例：「主隊勝率 58%」）
- [ ] 確認 AI 預測比分顯示（例：「預測 4-2」）
- [ ] 確認狀態是「排定中 / 進行中 / 已結束」其中之一

### 2. AI 詳情頁（點任一賽事卡片）

- [ ] 進入詳情頁有看到「AI 分析」文字區塊
- [ ] 看到雷達圖（六維度對比圖）
- [ ] 看到關鍵因素（4 個 bullet points）
- [ ] 看到分析摘要（200-400 字）
- [ ] 確認**沒有看到**「預測」「勝率」「命中率」這類詞（合規規範）
  - 應該是「推論」「推論機率」「驗證率」

### 3. 多日切換

- [ ] 切到「明天 06/23」tab
- [ ] 切到「後天 06/24」tab
- [ ] 確認不同日期的賽事列表都正常載入

### 4. 不同聯盟切換

- [ ] 切換到 NPB tab
- [ ] 切換到 CPBL tab
- [ ] 切換到 NBA tab
- [ ] 確認每個聯盟都有賽事（CPBL 6/22 沒賽事是正常的）

### 5. 離線 / 弱網

- [ ] 飛航模式開啟後，App 是否能優雅處理（顯示錯誤訊息而非當機）
- [ ] 重新連網後 pull-to-refresh 能否正常重抓

### 6. 廣告

- [ ] 賽事詳情頁底部有看到「廣告」區塊（測試版 AdMob 會顯示測試廣告）
- [ ] 確認廣告沒有擋到內容（不要擋到 AI 分析、比分、按鈕）
